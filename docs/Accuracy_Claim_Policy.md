# Accuracy Claim Policy

MediReport AI follows strict ethical standards regarding system capability claims.

## 1. Evidence-based Claims
All accuracy statements must be based on actual results from `ML_Model_Evaluation_Report.md`. Do not guess or exaggerate system performance.

## 2. Forbidden Claims
The following claims are strictly forbidden:
-   "100% accurate"
-   "Guaranteed diagnostic accuracy"
-   "Perfect extraction"
-   "Clinical-grade diagnosis"

## 3. Mandatory Context
Every accuracy claim must include the context of the evaluation set (e.g., "based on 500 synthetic test samples").

## 4. User Notification
If extraction confidence is below 75%, the app must display a "Manual Review Needed" warning.
